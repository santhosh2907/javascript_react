Box Model: content-box vs border-box

Both boxes have width: 200px, padding: 20px, and border: 10px.

content-box (Default)

200px + 40px + 20px = 260px width
Total Width: Width + Padding + Border.
The declared `width` only applies to the blue content area. Padding and border expand the element outwards.

border-box

200px Total Width
Total Width: Fixed at declared Width.
Padding and border eat into the content space. content width = 200 - 40 - 20 = 140px.
